# mp-upload

- 安装
  1. yarn [global] add @dima-toolbox/cli
  2. yarn [global] add @dima-toolbox/mp-upload

- options
  1. bScript构建的script指令，默认值 build
  2. distUrl构建后的文件夹目录（基于当前终端目录），默认值 dist
